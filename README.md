# IKEA Database Project

## Overview
This project is a relational database designed to model IKEA operations, including products, customers, sales, stores, and warehouses.

## Features
- Relational schema with primary and foreign keys
- Constraints: NOT NULL, UNIQUE, CHECK, DEFAULT
- Sample data for testing
- SQL queries for data analysis

## Files
- schema.sql → database structure (DDL)
- data.sql → data insertion (INSERT statements)
- queries.sql → queries for analysis (SELECT, UPDATE)
